### *Expose USB activity on public USB ports*


## What does my version change?
### In code,
I ported the code from the Arduino IDE to C/C++ and the Pi Pico SDK.
Now, about half a year later, I realize that the original creator cecio has done the same in the latest 1.0 release.

In my own process of porting the code, I dropped support for Pi-Watch and BadUSB as this was a personal project and I didn't need the features.

### In visuals,
I really liked the whole idea, but wanted to give the UI more customizability.

### More...
I plan on creating more versions, some of which might include a form of input on the device, but this is all unsure at the time I'm writing this.
